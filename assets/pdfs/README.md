## Creating a Good Resume PDF

1. **Format**: Save from Word/Google Docs as PDF
2. **File Size**: Compress if over 2MB
3. **Filename**: Use professional naming (FirstName_LastName_Resume.pdf)
4. **Content**: Keep to 1-2 pages
5. **Testing**: Verify it opens correctly

## Tools for PDF Creation

- Microsoft Word (Save As → PDF)
- Google Docs (File → Download → PDF)
- LaTeX (Overleaf for professional resumes)
- Canva (For creative resume designs)
- Adobe Acrobat (For advanced editing)

## Optimization

If your PDF is too large:
- Use [Smallpdf](https://smallpdf.com/compress-pdf)
- Or [iLovePDF](https://www.ilovepdf.com/compress_pdf)
- Target size: Under 2MB
