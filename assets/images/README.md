# Images Folder

Place your images here:

## Required Images

1. **profile.jpg** (600x600px recommended)
   - Your professional profile picture
   - Square format works best
   - Keep file size under 200KB

2. **favicon.png** (32x32px or 64x64px)
   - Browser tab icon
   - PNG format
   - Simple, recognizable design

## Optional Images

- Add any additional images for projects
- Use descriptive filenames (lowercase, hyphens instead of spaces)
- Optimize images before uploading (use TinyPNG.com)

## Image Guidelines

- **Format**: JPG for photos, PNG for graphics
- **Size**: Compress to balance quality and file size
- **Naming**: Use lowercase with hyphens (e.g., `my-project-screenshot.jpg`)
- **Dimensions**: Resize to actual display size before uploading
